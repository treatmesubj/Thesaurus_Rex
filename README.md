# Thesaurus Rex
It's a command-line thesaurus tool that uses thesaurus.com via HTTP requests, regular expressions, and JSON.

### Usage
cd/ into dir of the script or just put it on your system's PATH so you can use it from anywhere

`thesr.py [word]`
